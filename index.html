Questões sobre o git

Perguntas e respostas sobre o git

1-Quais os comandos para configurar usuário e e-mail no git?

Resposta: git config --global user.name

git config --global user.email

2-Qual a diferença entre git e github?

Resposta: O git é um sistema de controle de versão e o github é uma plataforma de hospedagem dos códigos fontes.

3- O que faz o comando commit?

Resposta: Atualiza as mudanças realizadas no código.

4- O que faz o comando push?

Resposta: Envia as informações de código para o github.

5-O que faz o comando pull?

Resposta: Busca as informações atualizadas do repositório.
